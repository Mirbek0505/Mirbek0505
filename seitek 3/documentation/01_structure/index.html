FAQ

How do I change the contact form information?

Open the PHP file in the pack, and replace the email address with yours. The PHP folder can be found only in the purchased version.

Page Structure

Attention! The page structure must be followed perfectyl for the item to work like it should! As the page runs REAL CSS3 transformations the code is a little more complex. We will explain it for everyone!

 

<div class="all-elements">
    <div id="perspective" class="perspective effect-airbnb"><!--this houses the entire page, and creates the effect-->
        <div class="header">
            <a href="#" id="showMenu"><i class="fa fa-navicon"></i></a>
            <a href="#" id="pageLogo"><img src="images/logo.png" alt="img"></a>
            <a href="#" id="openMail"><i class="fa fa-envelope-o"></i></a>
        </div>     
        <div class="outer-nav">
            <div class="inner-nav">
                <!-- NAVIGATION EXPLAIND IN NEXT CHAPTER --> 
            </div>
        </div>
        <div class="perspective_container"><!--the "moving to the left" content box-->
            <div class="wrapper"><!-- wrapper needed for scroll -->
                <div class="header-clear"><i class="fa fa-times"></i></div>
                <div class="content">
                    <div class="container">
                        Start Here!
                    </div>
                    <div class="decoration"></div>
                </div><!-- content-->
            </div><!-- wrapper -->
        </div><!-- /perspective container -->
    </div><!-- /perspective -->
</div><!-- /all elements wrapper -->

 

Line 1, the all-elements class, is create to ensure the page doesn't scroll left or right, this makes it overflow hidden so that all elements stay within the exact device screen widht

Line 2 the perspective wrapper, is created to house the two elements inside that will have animations. the effect class will be added and removed by JS when the nav is tapped.

Line 3 to 7 is the header. To change the page logo width, open style.css and locate pageLogo img and change the width value there.

Lin 8 to 12 will house the navigation. The outer nav is added to set top and left distance, and the inner nav is added to make the navigation list scrollable.

Line 13 is the box that actually moves to the left when you tap the menu. This will house all our content

Line 14 is the wrapper. Because line 13 is a fixed box, line 14 will create the scrolling proerty. Why is this needed? Because if line 13 would not be fixed, when the box would open, it would not be 80% in height, but the true hight of the element, so only a small gap would apear at the top, but the bottom would be scrollable.

Line 15 is the header clear class. The header is a fixed class, and it needs a spacing below it to clear.

Line 16 is the page content, this will house ALL the features in the template, added in contianers or responsive columns depending on what you need. This is the place where everything gets added in!

Lines 21 to 25 close all divs.

This is the most simple structure posible to make the item work flawlessly, altough many classes, the performance is greatly enhanced. We've included a starter-page for you go start directly if this is too much of a headache!

 

 

 

 

Navigation

In the previous chapter, in the page structure, in line 10, you can add the follwing classes to create menus

 

Simple Menu ( no dropdown )

This creates a simple navigation item. You can replace the i class with icons from Font Awesome http://fortawesome.github.io/Font-Awesome/icons/ and replace link and text with your needs.

<div class="nav-item">
    <a href="index.html" class="active-item">
        <i class="nav-icon fa fa-home"></i>
        Home
    </a>
</div>

Dropdown Menu ( innactive dropdown )

The dropdown menu is basically a simple menu, with another div in it that houses the submenu. The icon can be replaced like the main menu.

<div class="nav-item">
    <a href="#" class="deploy-nav-sub-item">
        <i class="nav-icon fa fa-cog"></i>
        Features
    </a>
    <div class="nav-sub-item">
        <a href="features-type.html">Type</a>
        <a href="features-jquery.html">jQuery</a>
        <a href="features-others.html">Others</a>
    </div>
</div>

Dropdown Menu  ( active dropdown )

The difference between an active menu and an inactive menu is that the active one has the class active-item added to the main anchor, and the class nav-sub-active added to the submenu div. To make a submenu item active, add the class active-item to any anchor from the submenu item.

<div class="nav-item">
    <a href="#" class="deploy-nav-sub-item active-item">
        <i class="nav-icon fa fa-cog"></i>
        Features
    </a>
    <div class="nav-sub-item nav-sub-active">
        <a class="active-item" href="features-type.html">Type</a>
        <a href="features-jquery.html">jQuery</a>
        <a href="features-others.html">Others</a>
    </div>
</div>

 

 

 

 

 

 

 


 

Homepage Slider

<div class="slider-container full-bottom">
    <div class="homepage-slider" data-snap-ignore="true">                
        <div>
            <div class="overlay"></div>
            <div class="homepage-slider-caption">
                <h3>Swipe me!</h3>
                <p>Hope you enjoy our latest item!</p>
            </div>
            <img src="images/pictures/3ww.jpg" class="responsive-image" alt="img">
        </div>
    </div>
</div>

The code above is the homepage slider. To create another slider, duplicate the div from line 3.

Line 4 is the overlay, the black background that allows the text to be more visibile

Line 4 is the homepage slider caption housing box.
Line 5 is the homepage slider title

Line 6 is the homepage slider text

Line 7 closes the caption

Line 8 is the image for the image slider.  We recommend using 16:9 format images.

Left Right Columns

<div class="one-half-responsive">
    <p class="thumb-left no-bottom">

<div class="decoration hide-if-responsive"></div>
<div class="one-half-responsive last-column">        
    <p class="thumb-right no-bottom">
        <img src="images/pictures/2s.jpg" alt="img">
        <strong>CSS3 Designed</strong>
        <em><br>Lorem Ipsum is simply dummy text of the printing and types. </em>
    </p>
    <div class="thumb-clear"></div>

Left right columns are colums that have one element to the left and one element to the right.

Line 1 creates the one-half-responsive class. This means the section will show 1 column on mobiles, and 2 on tablets.

Line 2 is the column itself. This is set to show to the left side of the screen.

Line 3 is the icon

Line 4 is the title

Line 5 is the text

Line 7 clears the floatation

Line 8 closes the lloating div.

Line 9 creates a decoration that hides itself on tablets.

Line 10 creates another one half responsive class

Line 11 creates a right column, the same princiles apply as the above.
